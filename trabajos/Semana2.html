Definiciones

Algunas de las definiciones de los temas que se abarcaron en la segunda semana fueron los siguientes:

Visual Studio Code: Es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Incluye soporte para depuración, control git incorporado, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactoring de código.

HTML (Lenguaje Marcado de Hipertexto): Es el lenguaje estándar para crear páginas y aplicaciones web. HTML proporciona la estructura de la página, utilizando etiquetas y atributos.

CSS (Hojas de Estilo en Cascada): Es un lenguaje de hojas de estilo que se utiliza para describir la apariencia de un documento escrito en HTML. CSS describe cómo se deben mostrar los elementos HTML en la pantalla.

Librerías CSS: Son colecciones de estilos predefinidos que puedes utilizar en tus proyectos para acelerar el desarrollo y garantizar la coherencia del diseño.
