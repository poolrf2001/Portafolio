Programación Front-End JavaScript, Lenguaje JavaScript

Durante la tercera semana, me centré en la programación front-end con JavaScript. Aprendí sobre la sintaxis de JavaScript, sus tipos de datos, como números, strings y booleanos, y cómo se usan para crear lógica y funcionalidad en las páginas web. Estudié los objetos en JavaScript, aprendiendo cómo se definen, cómo se usan y cómo se pueden modificar con métodos. También adquirí conocimientos sobre funciones y las más recientes funciones flecha, que proporcionan una sintaxis más corta y flexible. Además, exploré la creación de gráficos con JavaScript, utilizando librerías como Chart.js para representar datos de forma visual y dinámica.

Conceptos clave: JavaScript, Números, Strings, Objetos y Métodos, Booleanos, Funciones, Funciones Flecha y Gráficos

Definiciones

Algunas de las definiciones de los temas que se abarcaron en la segunda semana fueron los siguientes:

JavaScript: Es un lenguaje de programación que se utiliza principalmente para agregar interactividad y dinamismo a las páginas web en el lado del cliente (front-end).

Números, Strings y Booleanos: Son tipos de datos fundamentales en JavaScript. Los números representan valores numéricos, las strings representan secuencias de caracteres y los booleanos representan valores verdaderos o falsos.

Objetos y Métodos: En JavaScript, un objeto es una entidad independiente con propiedades y tipos. Un método es una función asociada a un objeto.

Funciones y Funciones Flecha: Una función es un bloque de código diseñado para realizar una tarea particular. Una función flecha es una forma más corta de declarar funciones en JavaScript, introducida en ES6.
