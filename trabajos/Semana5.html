Desarrollo Front-End con Framework JS, Gestión de Dependencias

Durante la quinta semana, me centré en el desarrollo front-end con frameworks de JavaScript, específicamente React.js. Aprendí sobre la estructura y operación de estos frameworks, y cómo utilizan el rendering en el lado del cliente para crear interfaces de usuario rápidas y eficientes. Estudié la gestión de dependencias, aprendiendo a utilizar herramientas como npm para instalar y administrar paquetes de software que mi proyecto necesita. También aprendí sobre las herramientas de creación de proyectos, como Create React App, que facilitan la configuración inicial de un nuevo proyecto. Además, realicé un laboratorio en el que configuré un nuevo proyecto y sus dependencias. Este laboratorio me permitió poner en práctica lo que había aprendido en un contexto práctico. Finalmente, exploré el desarrollo basado en componentes, aprendiendo a utilizar JSX para escribir elementos HTML en JavaScript, y a utilizar props y children para pasar datos entre componentes.

Conceptos clave: JavaScript, React, Jsx

Definiciones

Algunas de las definiciones de los temas que se abarcaron en la segunda semana fueron los siguientes:

Framework JS: Es un entorno de software que proporciona una estructura y un conjunto de herramientas para facilitar el desarrollo de aplicaciones en JavaScript.

JSX: Es una extensión de sintaxis para JavaScript que permite escribir HTML dentro de JavaScript.

PROPS: En React, las props son argumentos pasados a los componentes que permiten la comunicación entre ellos.

Gestión de Dependencias: Es el proceso de administrar las dependencias de software de un proyecto, incluyendo su instalación, actualización y configuración.

ACTIVIDADES DESARROLLADAS

Durante esta semana, instalé y configuré un nuevo proyecto con React utilizando Create React App y npm para gestionar las dependencias. Dividí la interfaz de usuario en varios componentes y utilicé JSX para escribir el marcado HTML de estos componentes en JavaScript. Utilicé props para pasar datos entre componentes y children para anidar componentes dentro de otros. Como resultado, creé una aplicación web funcional con React que utiliza un enfoque de desarrollo basado en componentes.
