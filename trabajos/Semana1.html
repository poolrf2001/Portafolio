Fundamentos de las tecnologías web, front-end, back-end, Mockups / UX Design

Durante esta semana, adquirí conocimientos fundamentales en una variedad de aspectos de las tecnologías web. Exploré los componentes básicos de las aplicaciones de Internet, estudiando cómo funcionan y cómo se comunican entre sí. Además, comprendí los conceptos fundamentales sobre front-end y back-end. En el lado de desarrollo web comprendí como nos permiten crear interfaces de usuario interactivas y atractivas. Mientras que en el lado del desarrollo back-end, comprendí el papel fundamental de los servidores, las bases de datos y las aplicaciones para proporcionar funcionalidad y gestionar los datos de las aplicaciones de Internet.

Conceptos clave: Fundamentos web, front-end,back-end

Definiciones

Algunas de las definiciones de los temas que se abarcaron en la primera semana fueron los siguientes:

Tecnologías web: Son todas aquellas herramientas y técnicas que permiten el desarrollo de sitios web y aplicaciones para Internet.

Desarrollo Front-end: Se refiere a la parte de la programación que involucra la interacción directa con el usuario. Usualmente involucra HTML, CSS y JavaScript.

Desarrollo Back-end: Es la parte de la programación que se encarga de que todo funcione detrás de escena. Este puede incluir el trabajo con servidores, bases de datos y aplicaciones.

Mockups / UX Design: Un mockup es un modelo a escala o tamaño completo de un diseño o dispositivo, utilizado para enseñar, demostrar, diseñar y evaluar. UX Design se refiere al proceso de crear productos que proporcionen experiencias significativas y relevantes para los usuarios.

ACTIVIDADES DESARROLLADAS

En esta sección identificaremos los procedimientos y resultados que obtuvimos Esta semana, revisé todos los fundamentos del desarrollo web, vi como está compuesto un sitio web. En el lado del front-end, vi algunas de las herramientas que se usan como HTML para estructurar el contenido, CSS para el estilo y JavaScript para añadir interactividad. En el back-end, conocí que Node.js sive para gestionar las solicitudes y respuestas del servidor y MongoDB funciona como base de datos. Como resultado, logré comprender la manera en que estos conceptos se relacionan y sirven de mucha ayuda al momento de tener que adentrarse cada vez más a este maravilloso mundo del desarrollo web.
