Eventos, Condicionales y Bucles con Framework JS, Renderizado Condicional/Iterativo, Formularios

Durante la sexta semana, me centré en el manejo de eventos, el renderizado condicional y el renderizado iterativo con frameworks de JavaScript, específicamente React.js. Aprendí cómo gestionar eventos de usuario, como clicks o entradas de texto, y cómo se puede manipular la interfaz de usuario en función de estos eventos. Estudié el renderizado condicional y iterativo, aprendiendo cómo se pueden mostrar u ocultar elementos en función de ciertas condiciones, y cómo se pueden mostrar listas de elementos. También aprendí sobre formularios en React, y cómo se pueden utilizar para recopilar y manejar la entrada de los usuarios. Además, exploré el uso de estilos en React, incluyendo estilos inline, hojas de estilo, módulos de estilo, y la utilización de componentes de estilo y frameworks CSS. Realicé un laboratorio en el que apliqué estos conceptos para mejorar la apariencia de mi proyecto.

Conceptos clave: Eventos y Renderizado Condicional e Iterativo, Utilización de Estilos en Framework JS, Styles Inline / Style Sheets / Styles Modules, Styles Components / Framework CSS

Definiciones

Algunas de las definiciones de los temas que se abarcaron en la segunda semana fueron los siguientes:

Eventos: Son las acciones que ocurren en la aplicación debido a la interacción del usuario o el sistema.

Renderizado Condicional/Iterativo: En React, el renderizado condicional es una forma de mostrar diferentes componentes o resultados basados en ciertas condiciones. El renderizado iterativo es una forma de mostrar una lista de componentes o resultados.

Formularios: En React, los formularios son una forma de recopilar y manejar la entrada de los usuarios.

Estilos en Framework JS: React permite diferentes formas de aplicar estilos a los componentes, incluyendo estilos inline, hojas de estilo, módulos de estilo, componentes de estilo y frameworks CSS.

ACTIVIDADES DESARROLLADAS

Durante esta semana, mejoré la interactividad de mi aplicación React al implementar la gestión de eventos. Utilicé el renderizado condicional para mostrar diferentes interfaces en función de las acciones del usuario, y el renderizado iterativo para mostrar listas de elementos. Incorporé formularios en mi aplicación para recopilar y manejar la entrada del usuario. También mejoré la apariencia de mi aplicación al aplicar estilos utilizando diferentes métodos, incluyendo estilos inline, hojas de estilo y módulos de estilo. Como resultado, logré crear una aplicación React que no sólo es funcional, sino también interactiva y visualmente atractiva.
