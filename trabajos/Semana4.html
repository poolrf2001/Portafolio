Métodos y Propiedades, Elementos, Estilos con JavaScript, Animaciones

Durante la cuarta semana, profundicé mi comprensión de los métodos y propiedades en JavaScript, y cómo se utilizan para interactuar con los elementos y estilos de las páginas web. Estudié cómo se pueden seleccionar y manipular elementos del DOM con JavaScript, y cómo se pueden cambiar los estilos de estos elementos para mejorar la apariencia y la interactividad de las páginas web. También exploré la creación de animaciones con JavaScript, aprendiendo a usar funciones de temporización como setTimeout y setInterval, y las APIs de animación web para crear animaciones suaves y atractivas.

Conceptos clave: Métodos y propiedades en JavaScript

Definiciones

Algunas de las definiciones de los temas que se abarcaron en la segunda semana fueron los siguientes:

Métodos y Propiedades: En JavaScript, las propiedades son valores asociados a un objeto, y los métodos son acciones que se pueden realizar en objetos.

Métodos y Propiedades: En JavaScript, las propiedades son valores asociados a un objeto, y los métodos son acciones que se pueden realizar en objetos.

Estilos con JavaScript: JavaScript puede cambiar todos los estilos CSS y todas las clases CSS de los elementos HTML.

Animaciones: Las animaciones son una forma de crear ilusiones de movimiento cambiando los estilos de los elementos en el tiempo.

ACTIVIDADES DESARROLLADAS

Como resultado de esta semana de aprendizaje, desarrollé un juego con JavaScript. Utilicé JavaScript para seleccionar y manipular elementos del DOM, cambiando sus estilos para crear una experiencia de juego interactiva y atractiva. Aproveché los métodos y propiedades de JavaScript para controlar el comportamiento del juego, y creé varias animaciones para hacer que el juego fuera más atractivo visualmente. Al final de la semana, había creado un juego funcional y atractivo utilizando únicamente JavaScript, HTML y CSS.
